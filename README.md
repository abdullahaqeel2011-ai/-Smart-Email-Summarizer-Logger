# 📬 Smart Email Summarizer & Logger

Automate Gmail email processing using n8n, AI summarization, and Google Sheets logging.

## Features

- Trigger workflow on incoming Gmail emails  
- Summarize and extract key information using AI (OpenAI / LangChain)  
- Append structured data to Google Sheets  
- Format date as `2 March 2026` for better readability  
- Fully automated workflow  

## Workflow Overview

1. **Gmail Trigger** – Listen for new emails in INBOX  
2. **AI Agent** – Summarize and extract fields in JSON format  
3. **Code Node** – Format date and prepare structured data  
4. **Google Sheets Node** – Append row with extracted data
